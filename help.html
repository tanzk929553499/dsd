This website is used to manage your prosthesis. The functions of each page are described below

Email Management

Set the contact email

Device Information

Provides the client with status information of the device

Administrator Function

Delete Cloud Information

Deletes everything of the specified device from the cloud

Model Page

In this page, you can manage the model in the Device and Cloud of the specific Algorithm. The specific functions are as follows:

Select Algorithm

Select an Algorithm

Select an Algorithm here, and after that the target of many following operations is the model of selected algorithm.

Update Model

Update Device Model

Acquire the device model of the selected algorithm form Cloud. Upload it for a specific algorithm to the Device and start to use selected algorithm

Train Model Using Collected Data

Train the device model of a specific algorithm with calibration data uploaded. You can acquire calibration data in DATA PAGE

Download Base Model

Download the base model of selected algorithm form Cloud

Download Model From Device

Download the current model of the current algorithm from the Device

Delete Model

Delete Local Model From Device

Clear device model of selected algorithm from the Device

Delete Model From Cloud

Clear device model of selected algorithm from the Cloud

Delete Model of All Algorithms

Clear device model of All algorithms from the Cloud

Data Page

In this page, you can manage the collection, transfer and cleanup of calibration data

Data Collection

Start Collection

When this button is pressed, the system will ask each uncollected action in sequence whether to collect now. When the user is ready, the data can be collected

Delete Local Data

Delete local data and the user can collect data again

Data Management

Upload Data

Upload new calibration data to the cloud platform

Delete Cloud Data

Clears calibration data from the cloud

Administrator Function

Download Cloud Data

(ADMIN ONLY)Download the calibration data from the cloud
